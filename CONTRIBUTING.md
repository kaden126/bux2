# Contributing
Contributions to `bux2` are always welcome! There are several ways you can help out.

## 1. Try it yourself
The easiest, and most effective thing you can do is use `bux2` for your own projects.
*Try things out!* Download `bux2` and
find things that are missing, things that don't work as expected (or at all),
or features you would like to have and open an issue.

## 2. Write code
If you feel confident enough, code your own solution to an issue, and create a pull request!
If the changes merge without breaking, and solve the problem, odds are they will be approved.
